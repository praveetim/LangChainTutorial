# LangChainTutorial
This is my personal tutorial for LangChain and LangGraph based on the book of "Learning LangChain Building AI and LLM Applications with LangChain and LangGraph" by Mayo Oshin &amp; Nuno Campos 

You need to create a file "api.env" and save the API key used in the project (e.g. OpenAI) in the same folder as the code.
